The Beginner's Guide to Hollywood:
A Survey of the Industry

Creating an infographic

Next

Project Details

  • Team: Tamanh Nguyen
  • Time: 8 Weeks
  • Tools Used: Adobe Illustrator, Microsoft Excel, Tableau

Overview

The goal of my Information Visualiziation and Aesthetics class was to create an infographic on a topic of our choice. Being an avid pop culture enthusiast, I set my sights on Hollywood.

Process

To begin with, we were told to select four infogaphics (models) that we wanted to emulate. We needed a base model for visual language, a typographic model, a color model, and a layout model. From there, we went through an iterative process in designing each of the data displays you see below. We would create a display, have it critiqued by our peers, and go through the process again.

For a more detailed description of the design process, please visit the portfolio page I set up in class: tamvn9.wix.com/tnguyen

Lessons Learned

  • Trust your instincts. Even if you may not feel like the best designer out there, you know what changes or considerations you need to make the most.
  • The pen tool in Adobe Illustrator is magic.
  • Don't forget the story in everything you do.
